# frontend

## Project setup
```
yarn install
```

### Environment settings (.env)
```
VUE_APP_GRAPHQL_WS=wss://test-api.meetjestad.net/v1/graphql
VUE_APP_GRAPHQL_TOKEN=...
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
